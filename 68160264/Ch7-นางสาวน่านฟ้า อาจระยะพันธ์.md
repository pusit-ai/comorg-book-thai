# บทที่ 7: ระบบอินพุต/เอาต์พุต (Input/Output)

เอกสารฉบับนี้สรุปเนื้อหาและโครงสร้างสถาปัตยกรรมระบบ I/O จากหนังสือ *Computer Organization and Architecture* เพื่อใช้เป็นคู่มือประกอบการศึกษาบน GitHub

---

## 1. อุปกรณ์ภายนอก (7.1 External Devices)
อุปกรณ์ภายนอกหรือ **Peripheral** เชื่อมต่อกับระบบผ่านโมดูล I/O แบ่งออกเป็น 3 ประเภทหลัก:
* **Human Readable:** ติดต่อกับผู้ใช้โดยตรง เช่น จอภาพ (Display), คีย์บอร์ด (Keyboard), เมาส์ (Mouse)
* **Machine Readable:** ติดต่อกับระบบประมวลผลหรือบันทึกข้อมูล เช่น ฮาร์ดดิสก์ (Hard Disk), SSD, เซนเซอร์
* **Communication:** สำหรับสื่อสารระยะไกล เช่น การ์ดเครือข่าย (NIC), โมเด็ม

## 2. โมดูลอินพุต/เอาต์พุต (7.2 I/O Modules)
ทำหน้าที่เป็นตัวกลางจัดการความแตกต่างระหว่างอุปกรณ์ภายนอกกับระบบภายใน
* **Control and Timing:** ควบคุมจังหวะการรับส่งข้อมูล
* **Processor Communication:** รับคำสั่งจาก CPU และรายงานสถานะ (Status Reporting)
* **Data Buffering:** พักข้อมูลชั่วคราวเพื่อชดเชยความเร็วที่แตกต่างกันระหว่างหน่วยความจำและอุปกรณ์

## 3. เทคนิคการดำเนินงาน I/O (7.3 - 7.5 I/O Techniques)
1.  **Programmed I/O:** CPU ควบคุมทุกขั้นตอนและต้องรอตรวจสอบสถานะอุปกรณ์ (**Polling**) ตลอดเวลา
2.  **Interrupt-Driven I/O:** CPU สั่งการแล้วไปทำงานอื่น เมื่ออุปกรณ์พร้อมจะส่งสัญญาณขัดจังหวะ (**Interrupt**) กลับมา
3.  **Direct Memory Access (DMA):** ใช้โมดูลพิเศษถ่ายโอนข้อมูลระหว่าง I/O และหน่วยความจำโดยตรง ไม่ผ่าน CPU

## 4. เทคโนโลยี Direct Cache Access (DCA) (7.6)
ออกแบบมาเพื่อเครือข่ายความเร็วสูง โดยอนุญาตให้โมดูล I/O ส่งข้อมูลเข้าสู่ **L3 Cache** ของ CPU โดยตรง เพื่อลดค่าความหน่วง (Latency)

## 5. ช่องทางการสื่อสารและตัวประมวลผล I/O (7.7 I/O Channels and Processors)
เมื่อระบบซับซ้อนขึ้น หน้าที่ I/O จะถูกส่งต่อไปยัง **I/O Channel** (มีชุดคำสั่งของตัวเอง) หรือ **I/O Processor** (มีหน่วยประมวลผลแยกอิสระ)

## 6. มาตรฐานการเชื่อมต่อภายนอก (7.8 External Interconnection Standards)
* **USB:** มาตรฐานสากลสำหรับอุปกรณ์ต่อพ่วง
* **SATA:** มาตรฐานสำหรับหน่วยเก็บข้อมูล (Storage)
* **PCI Express (PCIe):** บัสความเร็วสูงภายในเครื่อง
* **Thunderbolt:** การเชื่อมต่อประสิทธิภาพสูง รองรับทั้งข้อมูลและวิดีโอ

## 7. กรณีศึกษา: IBM zEnterprise EC12 I/O Structure (7.9)
ระบบเมนเฟรมที่ใช้ **Channel Subsystem (CSS)** เพื่อจัดการการรับส่งข้อมูลมหาศาลพร้อมกันอย่างมีเสถียรภาพสูง

## 8. คำสำคัญประจำบท (7.10 Key Terms)
* **Cycle Stealing:** DMA แย่งรอบบัสจาก CPU ชั่วคราวเพื่อย้ายข้อมูล
* **Interrupt Handler:** โปรแกรมจัดการเมื่อเกิดสัญญาณขัดจังหวะ
* **Peripheral Device:** อุปกรณ์ต่อพ่วงภายนอก
* **Data Buffering:** การพักข้อมูลเพื่อประสานความเร็ว
* **I/O Module:** หน่วยควบคุมตัวกลางระหว่าง CPU และอุปกรณ์ภายนอก
