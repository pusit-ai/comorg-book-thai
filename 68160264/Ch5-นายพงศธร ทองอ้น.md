Chapter 5 : Internal Memory
(หน่วยความจำภายใน)
หน่วยความจำภายในเป็นส่วนสำคัญของระบบคอมพิวเตอร์ เนื่องจากทำหน้าที่เก็บข้อมูลและโปรแกรมที่หน่วยประมวลผลกลาง (CPU) กำลังใช้งานอยู่ หน่วยความจำประเภทนี้เรียกว่า Main Memory หรือ Primary Memory ซึ่งแตกต่างจากหน่วยความจำสำรอง เช่น ฮาร์ดดิสก์หรือ SSD เพราะหน่วยความจำหลักมีความเร็วในการเข้าถึงข้อมูลสูงกว่าและสามารถทำงานร่วมกับ CPU ได้โดยตรง
ในอดีตคอมพิวเตอร์ใช้หน่วยความจำชนิด Magnetic Core Memory ซึ่งประกอบด้วยวงแหวนแม่เหล็กขนาดเล็กจำนวนมาก แต่เมื่อเทคโนโลยีอิเล็กทรอนิกส์พัฒนาขึ้น หน่วยความจำชนิดนี้จึงถูกแทนที่ด้วย Semiconductor Memory ซึ่งสร้างจากวงจรอิเล็กทรอนิกส์บนแผ่นชิป หน่วยความจำแบบเซมิคอนดักเตอร์มีข้อดีคือมีขนาดเล็กกว่า มีความเร็วสูงกว่า ใช้พลังงานน้อยกว่า และสามารถผลิตในปริมาณมากได้ง่าย
หน่วยพื้นฐานของหน่วยความจำเรียกว่า Memory Cell ซึ่งทำหน้าที่เก็บข้อมูลหนึ่งบิต แต่ละเซลล์สามารถเก็บค่าได้สองสถานะคือ 0 และ 1 เซลล์หน่วยความจำต้องมีคุณสมบัติสำคัญสามประการ คือ สามารถแสดงสถานะไบนารีสองค่าได้ สามารถเขียนข้อมูลลงไปในเซลล์ได้ และสามารถอ่านข้อมูลที่เก็บไว้ออกมาได้
เซลล์หน่วยความจำจะเชื่อมต่อกับวงจรอื่น ๆ ผ่านสายสัญญาณหลายชนิด โดยทั่วไปจะมีสายสำหรับเลือกเซลล์ที่ต้องการใช้งาน สายควบคุมสำหรับกำหนดว่าจะอ่านหรือเขียนข้อมูล และสายข้อมูลสำหรับรับหรือส่งข้อมูล เมื่อระบบต้องการเขียนข้อมูลลงในหน่วยความจำ จะมีการส่งสัญญาณไฟฟ้าไปยังเซลล์เพื่อกำหนดค่าเป็น 0 หรือ 1 ส่วนการอ่านข้อมูลจะเป็นการตรวจสอบสถานะของเซลล์นั้นว่ามีค่าเป็นอะไร
หน่วยความจำเซมิคอนดักเตอร์แบ่งออกเป็นสองประเภทหลัก คือ RAM (Random Access Memory) และ ROM (Read Only Memory) RAM เป็นหน่วยความจำที่สามารถอ่านและเขียนข้อมูลได้อย่างอิสระ คำว่า Random Access หมายถึงความสามารถในการเข้าถึงข้อมูลในตำแหน่งใด ๆ ของหน่วยความจำได้โดยใช้เวลาใกล้เคียงกันทุกตำแหน่ง RAM เป็นหน่วยความจำแบบ volatile ซึ่งหมายความว่าข้อมูลจะสูญหายเมื่อไม่มีไฟฟ้า
RAM สามารถแบ่งออกเป็นสองชนิดหลักคือ Dynamic RAM (DRAM) และ Static RAM (SRAM)
Dynamic RAM เป็นหน่วยความจำที่ใช้ตัวเก็บประจุไฟฟ้า หรือ capacitor ในการเก็บข้อมูล หนึ่งเซลล์ของ DRAM ประกอบด้วยทรานซิสเตอร์หนึ่งตัวและตัวเก็บประจุหนึ่งตัว เมื่อมีประจุไฟฟ้าอยู่ใน capacitor จะหมายถึงค่า 1 และเมื่อไม่มีประจุจะหมายถึงค่า 0 อย่างไรก็ตาม ประจุไฟฟ้าใน capacitor จะค่อย ๆ รั่วไหลออกไปตามเวลา ดังนั้น DRAM จึงต้องมีการ refresh หรือการเติมประจุไฟฟ้าใหม่ให้กับเซลล์หน่วยความจำเป็นระยะเพื่อรักษาข้อมูลเอาไว้
ข้อดีของ DRAM คือมีโครงสร้างที่เรียบง่าย ทำให้สามารถบรรจุเซลล์หน่วยความจำจำนวนมากลงในชิปเดียวได้ ส่งผลให้มีความจุสูงและราคาถูกกว่า SRAM ด้วยเหตุนี้ DRAM จึงถูกใช้เป็นหน่วยความจำหลักของคอมพิวเตอร์ อย่างไรก็ตาม DRAM มีข้อเสียคือมีความเร็วในการทำงานต่ำกว่า SRAM และต้องมีวงจรสำหรับการ refresh อยู่เสมอ
Static RAM หรือ SRAM ใช้โครงสร้างวงจรที่เรียกว่า Flip-Flop ในการเก็บข้อมูล ซึ่งประกอบด้วยทรานซิสเตอร์หลายตัว โดยทั่วไปหนึ่งเซลล์ของ SRAM ใช้ทรานซิสเตอร์ประมาณหกตัวในการสร้างวงจรที่สามารถเก็บค่าบิตได้อย่างเสถียร ข้อดีของ SRAM คือไม่จำเป็นต้อง refresh ข้อมูลเหมือน DRAM และสามารถเข้าถึงข้อมูลได้รวดเร็วกว่า
เนื่องจาก SRAM มีความเร็วสูง จึงมักถูกนำมาใช้เป็น Cache Memory ซึ่งเป็นหน่วยความจำขนาดเล็กที่อยู่ใกล้กับ CPU และทำหน้าที่เก็บข้อมูลที่ถูกใช้งานบ่อยเพื่อลดเวลาในการเข้าถึงข้อมูลจากหน่วยความจำหลัก อย่างไรก็ตาม SRAM มีราคาสูงและมีความหนาแน่นของข้อมูลต่ำกว่า DRAM จึงไม่เหมาะสำหรับใช้เป็นหน่วยความจำหลักที่ต้องมีขนาดใหญ่
อีกประเภทหนึ่งของหน่วยความจำคือ ROM (Read Only Memory) ซึ่งเป็นหน่วยความจำที่ข้อมูลถูกบันทึกไว้ถาวรและไม่สูญหายเมื่อไฟฟ้าถูกตัดออก ROM จึงถูกเรียกว่าเป็นหน่วยความจำแบบ nonvolatile โดยทั่วไป ROM ใช้เก็บโปรแกรมระบบที่จำเป็นต่อการเริ่มต้นการทำงานของคอมพิวเตอร์ เช่น โปรแกรมใน BIOS หรือ firmware ของอุปกรณ์ต่าง ๆ
ROM มีหลายประเภทที่พัฒนาขึ้นตามความต้องการในการใช้งาน ประเภทแรกคือ ROM แบบดั้งเดิมซึ่งข้อมูลถูกกำหนดไว้ตั้งแต่ขั้นตอนการผลิตของโรงงาน ทำให้ไม่สามารถเปลี่ยนแปลงข้อมูลได้ภายหลัง ต่อมามีการพัฒนา PROM (Programmable ROM) ซึ่งผู้ใช้งานสามารถเขียนข้อมูลลงไปได้ครั้งเดียวโดยใช้เครื่องโปรแกรมพิเศษ
จากนั้นจึงมีการพัฒนา EPROM (Erasable Programmable ROM) ซึ่งสามารถลบข้อมูลได้โดยใช้แสงอัลตราไวโอเลต ก่อนจะเขียนข้อมูลใหม่ลงไป อย่างไรก็ตามการลบข้อมูลต้องลบทั้งชิป ทำให้การใช้งานค่อนข้างยุ่งยาก เทคโนโลยีต่อมาคือ EEPROM (Electrically Erasable Programmable ROM) ซึ่งสามารถลบข้อมูลได้ด้วยไฟฟ้าและสามารถลบเฉพาะบางส่วนของหน่วยความจำได้
เทคโนโลยีที่ได้รับความนิยมมากในปัจจุบันคือ Flash Memory ซึ่งเป็นรูปแบบหนึ่งของ EEPROM แต่มีความสามารถในการลบข้อมูลเป็นกลุ่มหรือ block ทำให้การทำงานรวดเร็วขึ้น Flash Memory ถูกใช้งานอย่างแพร่หลายในอุปกรณ์จัดเก็บข้อมูลสมัยใหม่ เช่น USB Flash Drive, Memory Card และ Solid State Drive (SSD)
ในระหว่างการทำงานของหน่วยความจำ อาจเกิดข้อผิดพลาดของข้อมูลได้ ซึ่งข้อผิดพลาดเหล่านี้สามารถแบ่งออกเป็นสองประเภทคือ Hard Error และ Soft Error Hard Error เกิดจากความเสียหายของฮาร์ดแวร์ เช่น เซลล์หน่วยความจำเสียหรือวงจรมีปัญหา ส่วน Soft Error เป็นข้อผิดพลาดชั่วคราวที่เกิดจากสภาพแวดล้อม เช่น สัญญาณรบกวนทางไฟฟ้าหรือรังสีจากธรรมชาติที่อาจทำให้ค่าของบิตเปลี่ยนแปลง
เพื่อป้องกันและแก้ไขข้อผิดพลาดของข้อมูล ระบบหน่วยความจำจึงใช้เทคนิคที่เรียกว่า Error Correcting Code (ECC) หลักการของ ECC คือการเพิ่มบิตพิเศษที่เรียกว่า check bits เข้าไปพร้อมกับข้อมูล เมื่อมีการอ่านข้อมูลจากหน่วยความจำ ระบบจะคำนวณค่า check bits ใหม่และเปรียบเทียบกับค่าที่บันทึกไว้ หากพบว่ามีความแตกต่างก็สามารถตรวจพบข้อผิดพลาดและในบางกรณีสามารถแก้ไขข้อมูลที่ผิดพลาดได้
หนึ่งในเทคนิค ECC ที่สำคัญคือ Hamming Code ซึ่งเป็นวิธีที่ใช้บิตตรวจสอบหลายบิตเพื่อระบุตำแหน่งของบิตที่เกิดข้อผิดพลาด วิธีนี้สามารถแก้ไขข้อผิดพลาดที่เกิดขึ้นเพียงหนึ่งบิตได้ และสามารถตรวจจับข้อผิดพลาดหลายบิตได้ในบางกรณี ระบบที่ใช้เทคนิคนี้มักเรียกว่า SEC (Single Error Correction) หรือ SEC-DED (Single Error Correction Double Error Detection)
นอกจากนี้ยังมีการพัฒนาเทคโนโลยีหน่วยความจำ DRAM ให้มีประสิทธิภาพสูงขึ้น เช่น SDRAM (Synchronous DRAM) ซึ่งทำงานสอดคล้องกับสัญญาณ clock ของระบบ ทำให้สามารถประสานการทำงานกับ CPU ได้ดีขึ้น เทคโนโลยีที่พัฒนาต่อมาคือ DDR DRAM (Double Data Rate DRAM) ซึ่งสามารถส่งข้อมูลได้สองครั้งในหนึ่งรอบของสัญญาณ clock โดยส่งข้อมูลทั้งในช่วงขอบขึ้นและขอบลงของสัญญาณ clock ทำให้มีอัตราการส่งข้อมูลสูงกว่า SDRAM แบบเดิม
เทคโนโลยี DDR ได้พัฒนาอย่างต่อเนื่องจนเกิดมาตรฐานหลายรุ่น เช่น DDR, DDR2, DDR3, DDR4 และ DDR5 ซึ่งแต่ละรุ่นมีความเร็วสูงขึ้น ใช้พลังงานต่ำลง และรองรับความจุที่มากขึ้น
Flash Memory ก็เป็นเทคโนโลยีหน่วยความจำที่สำคัญอีกประเภทหนึ่ง Flash Memory ใช้ทรานซิสเตอร์ชนิดพิเศษที่เรียกว่า Floating Gate Transistor ในการเก็บข้อมูล อิเล็กตรอนจะถูกเก็บไว้ใน floating gate ซึ่งสามารถรักษาประจุไฟฟ้าไว้ได้แม้ไม่มีแหล่งจ่ายไฟ ทำให้ข้อมูลไม่สูญหายเมื่อปิดเครื่อง
Flash Memory มีสองรูปแบบหลักคือ NOR Flash และ NAND Flash NOR Flash มีความสามารถในการเข้าถึงข้อมูลแบบสุ่มหรือ random access และมีความเร็วในการอ่านสูง จึงเหมาะสำหรับใช้ในระบบ embedded หรืออุปกรณ์ที่ต้องการอ่านโปรแกรมโดยตรง ส่วน NAND Flash มีความหนาแน่นของข้อมูลสูงและมีความเร็วในการเขียนข้อมูลสูง จึงถูกใช้ในอุปกรณ์จัดเก็บข้อมูล เช่น SSD, USB Drive และ Memory Card
ในปัจจุบันนักวิจัยกำลังพัฒนาเทคโนโลยีหน่วยความจำแบบใหม่ที่สามารถรวมข้อดีของ DRAM และ Flash เข้าด้วยกัน หนึ่งในเทคโนโลยีเหล่านี้คือ STT-RAM (Spin Transfer Torque RAM) ซึ่งใช้คุณสมบัติของสนามแม่เหล็กในการเก็บข้อมูล ทำให้สามารถเก็บข้อมูลได้โดยไม่ต้องใช้พลังงานไฟฟ้าอย่างต่อเนื่อง
อีกเทคโนโลยีหนึ่งคือ PCRAM (Phase Change RAM) ซึ่งใช้วัสดุชนิดพิเศษที่สามารถเปลี่ยนสถานะระหว่างโครงสร้างผลึกและโครงสร้างไม่เป็นระเบียบ การเปลี่ยนสถานะนี้ทำให้ค่าความต้านทานของวัสดุเปลี่ยนไป จึงสามารถใช้แทนค่า 0 และ 1 ได้
นอกจากนี้ยังมี ReRAM (Resistive RAM) ซึ่งใช้การเปลี่ยนแปลงค่าความต้านทานของวัสดุเมื่อมีกระแสไฟฟ้าไหลผ่าน เทคโนโลยีเหล่านี้มีศักยภาพที่จะกลายเป็นหน่วยความจำรุ่นใหม่ที่มีความเร็วสูง ใช้พลังงานต่ำ และสามารถเก็บข้อมูลได้แม้ไม่มีไฟฟ้า
หน่วยความจำภายในจึงเป็นองค์ประกอบสำคัญของระบบคอมพิวเตอร์ เพราะมีผลโดยตรงต่อความเร็วและประสิทธิภาพของการประมวลผล การพัฒนาเทคโนโลยีหน่วยความจำอย่างต่อเนื่องช่วยให้คอมพิวเตอร์สามารถทำงานได้เร็วขึ้น รองรับข้อมูลจำนวนมากขึ้น และมีความน่าเชื่อถือสูงขึ้นในอนาคต
