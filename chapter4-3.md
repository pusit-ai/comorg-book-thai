4.3 องค์ประกอบของการออกแบบแคช (Elements of Cache Design)
| องค์ประกอบ | รายละเอียด |
| :--- | :--- |
| ขนาดแคช (Cache Size) | ต้องไม่เล็กเกินจนประสิทธิภาพต่ำ และไม่ใหญ่เกินจนราคาแพง |
| ฟังก์ชันการจับคู่ (Mapping) | Direct, Associative, Set Associative |
| อัลกอริทึมการแทนที่ | LRU (ใช้น้อยสุด), FIFO, LFU, Random |
| นโยบายการเขียน (Write Policy) | Write-through (เขียนพร้อมกัน), Write-back (เขียนทีหลัง) |
| ขนาดบรรทัด (Line Size) | ขนาดของบล็อกข้อมูลที่ดึงมา |
| จำนวนของแคช | ระดับเดียว (Single) หรือหลายระดับ (L1, L2, L3) |
