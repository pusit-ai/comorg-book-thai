# บทที่ 5 หน่วยความจำภายใน (Internal Memory)
## วัตถุประสงค์การเรียนรู้
เมื่อศึกษาบทนี้จบแล้ว ผู้อ่านควรมีความสามารถดังต่อไปนี้
- อธิบายภาพรวมของหน่วยความจำหลักชนิดสารกึ่งตัวนำประเภทต่าง ๆ ที่สำคัญได้
- เข้าใจการทำงานของรหัสพื้นฐานที่ใช้ในการตรวจจับและแก้ไขข้อผิดพลาดแบบบิตเดียว (Single-bit errors) ในคำขนาด 8 บิต
- สรุปคุณสมบัติของโครงสร้างหน่วยความจำ DDR DRAM ในปัจจุบันได้
- เข้าใจความแตกต่างระหว่างหน่วยความจำแฟลชแบบ NOR และ NAND
- อธิบายภาพรวมของเทคโนโลยีหน่วยความจำสถานะของแข็งแบบไม่ลบเลือน (Nonvolatile solid-state memory) รุ่นใหม่ได้

## 5.1 หน่วยความจำหลักชนิดสารกึ่งตัวนำ (SEMICONDUCTOR MAIN MEMORY)
ในระบบคอมพิวเตอร์ยุคแรก รูปแบบของหน่วยความจำเข้าถึงโดยสุ่ม (Random-access storage) ที่นิยมใช้มากที่สุดสำหรับหน่วยความจำหลัก คือการใช้แถวลำดับของวงแหวนแม่เหล็กเฟอร์โรต์ที่มีลักษณะคล้ายโดนัท ซึ่งเรียกว่า **คอร์ (Cores)** ด้วยเหตุนี้ หน่วยความจำหลักจึงมักถูกเรียกว่า "คอร์" มาจนถึงปัจจุบัน อย่างไรก็ตาม การอุบัติขึ้นและข้อดีของเทคโนโลยีไมโครอิเล็กทรอนิกส์ได้เข้ามาแทนที่หน่วยความจำคอร์แม่เหล็กไปนานแล้ว ในปัจจุบันการใช้ชิปสารกึ่งตัวนำสำหรับหน่วยความจำหลักถือเป็นมาตรฐานสากล ซึ่งประเด็นสำคัญของเทคโนโลยีนี้จะถูกนำเสนอในหัวข้อนี้
### โครงสร้าง (Organization)
องค์ประกอบพื้นฐานของ **หน่วยความจำชนิดสารกึ่งตัวนำ (Semiconductor memory)** คือ เซลล์หน่วยความจำ (Memory cell) แม้จะมีการใช้เทคโนโลยีอิเล็กทรอนิกส์ที่หลากหลาย แต่เซลล์หน่วยความจำชนิดสารกึ่งตัวนำทั้งหมดจะมีคุณสมบัติร่วมกันบางประการ ดังนี้
- แสดงสถานะที่เสถียร (หรือกึ่งเสถียร) ได้สองสถานะ ซึ่งสามารถใช้แทนค่าเลขฐานสอง 1 และ 0
- สามารถเขียนข้อมูลลงไปได้อย่างน้อยหนึ่งครั้งเพื่อกำหนดสถานะ
- สามารถอ่านข้อมูลเพื่อรับทราบสถานะที่บันทึกไว้ได้

ทำงานของเซลล์หน่วยความจำ โดยทั่วไปเซลล์จะมีขั้วหน้าที่สามขั้วที่สามารถนำสัญญาณไฟฟ้าได้ ได้แก่ **ขั้วเลือก (Select terminal)** ทำหน้าที่เลือกเซลล์หน่วยความจำสำหรับการดำเนินการอ่านหรือเขียน **ขั้วควบคุม (Control terminal)** ทำหน้าที่ระบุประเภทการทำงานว่าเป็น "อ่าน" หรือ "เขียน"

สำหรับการเขียน ขั้วอื่น ๆ จะส่งสัญญาณไฟฟ้าเพื่อกำหนดสถานะของเซลล์ให้เป็น 1 หรือ 0 ส่วนการอ่าน ขั้วดังกล่าวจะทำหน้าที่ส่งสัญญาณแสดงสถานะของเซลล์ออกมา ทั้งนี้ รายละเอียดเชิงลึกเกี่ยวกับโครงสร้างภายใน การทำงาน และจังหวะเวลา (Timing) ของเซลล์หน่วยความจำจะขึ้นอยู่กับเทคโนโลยีวงจรรวมเฉพาะทางที่ใช้ ซึ่งเกินขอบเขตของหนังสือเล่มนี้ (ยกเว้นในส่วนของสรุปย่อ) โดยในที่นี้ เราจะกำหนดสมมติฐานพื้นฐานว่าแต่ละเซลล์สามารถถูกเลือกเพื่อดำเนินการอ่านและเขียนข้อมูลได้
### DRAM และ SRAM
หน่วยความจำทุกประเภทที่จะกล่าวถึงในบทนี้เป็นแบบเข้าถึงโดยสุ่ม กล่าวคือ ข้อมูลแต่ละคำในหน่วยความจำสามารถเข้าถึงได้โดยตรงผ่านตรรกะการระบุตำแหน่ง (Addressing logic) ที่เชื่อมต่ออยู่ภายใน

ประเภทหลักของหน่วยความจำสารกึ่งตัวนำ ประเภทที่พบได้บ่อยที่สุดมักถูกเรียกว่า **หน่วยความจำเข้าถึงโดยสุ่ม หรือ แรม (RAM: Random-access memory)** ซึ่งในความเป็นจริงแล้วถือเป็นการใช้คำที่คลาดเคลื่อน เนื่องจากหน่วยความจำทุกประเภทในตารางต่างก็เป็นแบบเข้าถึงโดยสุ่มเช่นกัน อย่างไรก็ตาม ลักษณะเฉพาะที่สำคัญของหน่วยความจำที่เรียกว่า RAM คือความสามารถในการอ่านข้อมูลจากหน่วยความจำและเขียนข้อมูลใหม่ลงไปได้อย่างสะดวกและรวดเร็ว โดยทั้งการอ่านและการเขียนจะกระทำผ่านสัญญาณไฟฟ้า

อีกลักษณะเฉพาะหนึ่งของ RAM แบบดั้งเดิมคือความสามารถในการลบเลือนข้อมูลได้เมื่อไม่มีไฟฟ้าเลี้ยง (Volatile) โดย RAM จำเป็นต้องได้รับพลังงานไฟฟ้าอย่างต่อเนื่อง หากไฟฟ้าขัดข้องข้อมูลจะสูญหายทันที ดังนั้น RAM จึงถูกใช้เพื่อเป็นหน่วยจัดเก็บข้อมูลชั่วคราวเท่านั้น รูปแบบดั้งเดิมของ RAM สองประเภทที่ใช้ในคอมพิวเตอร์คือ DRAM และ SRAM ส่วน RAM รูปแบบใหม่ ๆ ที่จะกล่าวถึงในหัวข้อ 5.5 นั้น จะเป็นแบบไม่ลบเลือนข้อมูล (Nonvolatile)

***ไดนามิกแรม (DYNAMIC RAM)***
เทคโนโลยี RAM แบ่งออกเป็นสองประเภทหลัก ได้แก่ แบบไดนามิก (Dynamic) และแบบสแตติก (Static) สำหรับ ไดนามิกแรม (DRAM) นั้นถูกสร้างขึ้นจากเซลล์ที่เก็บข้อมูลในรูปแบบของประจุไฟฟ้าภายในตัวเก็บประจุ (Capacitors) การมีหรือไม่มีประจุไฟฟ้าจะถูกตีความว่าเป็นค่าเลขฐานสอง 1 หรือ 0 เนื่องจากตัวเก็บประจุมีแนวโน้มตามธรรมชาติที่จะคายประจุ DRAM จึงจำเป็นต้องมีการกระตุ้นประจุใหม่ (Charge refreshing) เป็นระยะ เพื่อรักษาข้อมูลที่จัดเก็บไว้ คำว่า ไดนามิก จึงหมายถึงแนวโน้มที่ประจุไฟฟ้าที่เก็บไว้จะรั่วไหลออกไปแม้ว่าจะมีการจ่ายไฟอย่างต่อเนื่องก็ตาม

โครงสร้าง DRAM ทั่วไปสำหรับหนึ่งเซลล์ที่เก็บข้อมูลหนึ่งบิต สายระบุตำแหน่ง (Address line) จะถูกกระตุ้นเมื่อต้องการอ่านหรือเขียนค่าบิตจากเซลล์นี้ ทรานซิสเตอร์จะทำหน้าที่เป็นสวิตช์ที่จะปิดวงจร (ยอมให้กระแสไหล) หากมีการจ่ายแรงดันไฟฟ้าไปยังสายระบุตำแหน่ง และเปิดวงจร (ไม่ยอมให้กระแสไหล) หากไม่มีแรงดันไฟฟ้า
- การดำเนินการเขียนจะมีการจ่ายสัญญาณแรงดันไฟฟ้าไปยังสายบิต (Bit line) โดยแรงดันไฟฟ้าสูงแทนค่า 1 และแรงดันไฟฟ้าต่ำแทนค่า 0 จากนั้นจะมีการจ่ายสัญญาณไปยังสายระบุตำแหน่งเพื่อยอมให้ประจุไฟฟ้าถ่ายโอนไปยังตัวเก็บประจุ
- การดำเนินการอ่านเมื่อเลือกสายระบุตำแหน่ง ทรานซิสเตอร์จะทำงานและประจุที่เก็บไว้ในตัวเก็บประจุจะถูกส่งออกไปยังสายบิตเพื่อเข้าสู่เครื่องขยายสัญญาณ (Sense amplifier) ซึ่งจะเปรียบเทียบแรงดันไฟฟ้าของตัวเก็บประจุกับค่าอ้างอิงเพื่อตัดสินว่าเซลล์นั้นเก็บค่าตรรกะ 1 หรือ 0 การอ่านข้อมูลจะทำให้ประจุในตัวเก็บประจุหมดไป ดังนั้นจึงต้องมีการประจุไฟกลับคืน (Restore) เพื่อให้การดำเนินการเสร็จสมบูรณ์

แม้ว่าเซลล์ DRAM จะใช้เพื่อเก็บข้อมูลเพียงบิตเดียว (0 หรือ 1) แต่อุปกรณ์นี้มีลักษณะเป็นอะนาล็อก (Analog) เนื่องจากตัวเก็บประจุสามารถเก็บค่าประจุในช่วงใดก็ได้ โดยจะมีค่าขีดเริ่มเปลี่ยน (Threshold) เป็นตัวกำหนดว่าจะตีความประจุนั้นเป็น 1 หรือ 0

***สแตติกแรม (STATIC RAM)*** ในทางตรงกันข้าม สแตติกแรม (SRAM) เป็นอุปกรณ์ดิจิทัลที่ใช้องค์ประกอบตรรกะแบบเดียวกับที่ใช้ในตัวประมวลผล ใน SRAM ค่าเลขฐานสองจะถูกจัดเก็บโดยใช้โครงสร้างลอจิกเกตแบบฟลิปฟลอป (Flip-flop) โดย SRAM จะเก็บรักษาข้อมูลไว้ได้นานเท่าที่มีการจ่ายกระแสไฟฟ้าให้

***การเปรียบเทียบระหว่าง SRAM และ DRAM*** หน่วยความจำประเภท Static RAM (SRAM) และ Dynamic RAM (DRAM) จัดเป็นหน่วยความจำประเภทลบเลือนได้ (volatile memory) กล่าวคือ จำเป็นต้องมีการจ่ายกระแสไฟฟ้าอย่างต่อเนื่องเพื่อรักษาค่าข้อมูลในหน่วยความจำ เซลล์หน่วยความจำแบบไดนามิก มีโครงสร้างที่เรียบง่ายและมีขนาดเล็กกว่าเซลล์หน่วยความจำแบบคงที่ ด้วยเหตุนี้ DRAM จึงมีความหนาแน่นของข้อมูลสูงกว่า (เซลล์ขนาดเล็กกว่าหมายถึงสามารถบรรจุจำนวนเซลล์ได้มากขึ้นต่อหนึ่งหน่วยพื้นที่) และมีต้นทุนการผลิตที่ต่ำกว่า SRAM ในระดับความจุเดียวกัน อย่างไรก็ตาม DRAM จำเป็นต้องมีวงจรสำหรับรีเฟรชข้อมูล (refresh circuitry) เข้ามาสนับสนุน สำหรับหน่วยความจำที่มีขนาดใหญ่ ต้นทุนคงที่ของวงจรรีเฟรชนี้จะถูกชดเชยด้วยต้นทุนผันแปรที่ต่ำกว่าของเซลล์ DRAM ทำให้ DRAM มักเป็นตัวเลือกที่เหมาะสมกว่าสำหรับความต้องการหน่วยความจำขนาดใหญ่ นอกจากนี้ ประเด็นสำคัญประการสุดท้ายคือ SRAM มีความเร็วในการทำงานที่สูงกว่า DRAM เล็กน้อย ด้วยคุณสมบัติที่แตกต่างกันเหล่านี้ SRAM จึงถูกเลือกใช้งานเป็นหน่วยความจำแคช (ทั้งแบบที่อยู่ภายในและภายนอกชิปประมวลผล) ในขณะที่ DRAM จะถูกเลือกใช้เป็นหน่วยความจำหลักของระบบ

### ประเภทของหน่วยความจำแบบอ่านอย่างเดียว (ROM)
หน่วยความจำแบบอ่านอย่างเดียว (Read-Only Memory: ROM) เป็นหน่วยความจำประเภทไม่ลบเลือน (nonvolatile) ซึ่งไม่จำเป็นต้องใช้แหล่งจ่ายไฟเพื่อรักษาข้อมูลไว้ โดยข้อมูลจะถูกบันทึกไว้เป็นการถาวรตั้งแต่กระบวนการผลิต

**ประเภทของ ROM และหน่วยความจำแบบอ่านเป็นส่วนใหญ่**

- ROM: ข้อมูลจะถูกเชื่อมต่อลงในชิปโดยตรงในระหว่างกระบวนการผลิต ซึ่งมีต้นทุนคงที่สูงและไม่มีโอกาสแก้ไขหากเกิดข้อผิดพลาด
- PROM (Programmable ROM): เป็นทางเลือกที่คุ้มค่ากว่าเมื่อต้องการจำนวนไม่มาก โดยสามารถเขียนข้อมูลลงไปได้เพียงครั้งเดียวด้วยอุปกรณ์พิเศษ
- EPROM (Erasable PROM): สามารถอ่านและเขียนได้ด้วยไฟฟ้า แต่ต้องลบข้อมูลเดิมออกทั้งหมดด้วยรังสีอัลตราไวโอเลตก่อนการเขียนใหม่ ซึ่งใช้เวลานาน
- EEPROM (Electrically Erasable PROM): สามารถเขียนทับข้อมูลได้โดยไม่ต้องลบข้อมูลเดิมก่อน และสามารถแก้ไขข้อมูลเฉพาะไบต์ที่ต้องการได้ แต่มีความซับซ้อนและราคาสูงกว่า
- Flash Memory: มีลักษณะการทำงานกึ่งกลางระหว่าง EPROM และ EEPROM โดยใช้เทคโนโลยีการลบข้อมูลด้วยไฟฟ้าที่รวดเร็ว (ลบเป็นบล็อก) และมีความหนาแน่นของข้อมูลสูง

### ตรรกะของชิป (Chip Logic)

หน่วยความจำสารกึ่งตัวนำถูกผลิตและบรรจุมาในรูปแบบของชิปวงจรรวม (Integrated Circuit Chips) โดยภายในแต่ละชิปจะประกอบด้วยชุดของเซลล์หน่วยความจำที่จัดเรียงตัวเป็นอาเรย์ (Array)

ในการออกแบบระบบลำดับชั้นของหน่วยความจำ (Memory Hierarchy) ผู้ออกแบบต้องพิจารณาความสมดุลระหว่างความเร็ว ความหนาแน่นของข้อมูล และต้นทุน ซึ่งปัจจัยเหล่านี้ส่งผลโดยตรงต่อการกำหนดโครงสร้างตรรกะและการจัดวางเซลล์หน่วยความจำบนชิป ประเด็นสำคัญในการออกแบบคือการกำหนดจำนวนบิตของข้อมูลที่จะทำการอ่านหรือเขียนในแต่ละครั้ง รูปแบบการจัดโครงสร้างมีตั้งแต่การจัดเรียงเซลล์ในอาเรย์ให้ตรงกับเวิร์ดเชิงตรรกะ (Logical Words) เช่น ชิปขนาด 16 เมกะบิต อาจจัดโครงสร้างเป็น 1 ล้านเวิร์ด เวิร์ดละ 16 บิต ไปจนถึงการจัดโครงสร้างแบบอ่านเขียนครั้งละ 1 บิต (1-bit-per-chip organization)

**โครงสร้างภายในของชิปหน่วยความจำ** ตัวอย่างเช่น ในชิปหน่วยความจำ DRAM ขนาด 16 เมกะบิต ที่มีการอ่านหรือเขียนข้อมูลครั้งละ 4 บิต อาเรย์ของหน่วยความจำจะถูกจัดแบ่งเป็นอาเรย์สี่เหลี่ยมจัตุรัสขนาด 2,048 x 2,048 องค์ประกอบ โดยแต่ละองค์ประกอบจะเชื่อมต่อกับสายสัญญาณแนวนอนและแนวตั้ง ดังนี้
- สายสัญญาณแถว (Row lines): เชื่อมต่อกับขั้วเลือก (Select terminal) ของทุกเซลล์ในแถวนั้นๆ
- สายสัญญาณคอลัมน์ (Column lines): เชื่อมต่อกับขั้วข้อมูล/การตรวจรู้ (Data-In/Sense terminal) ของทุกเซลล์ในคอลัมน์นั้นๆ

**การถอดรหัสที่อยู่และการรับส่งข้อมูล (Addressing and Data I/O)** ในการเลือกเวิร์ดที่ต้องการ จะมีการใช้ สายสัญญาณที่อยู่ (Address lines) จำนวน log<sub>2</sub>W เส้น จากตัวอย่างข้างต้น จะใช้สายสัญญาณที่อยู่ 11 เส้นเพื่อป้อนเข้าสู่ ตัวถอดรหัสแถว (Row Decoder) ซึ่งจะเปลี่ยนสัญญาณขาเข้า 11 เส้น ให้เป็นการเลือกเอาต์พุต 1 สายจากทั้งหมด 2,048 สาย ตามรูปแบบบิตที่ได้รับ (2<sup>11</sup>=2,048)

ในกระบวนการอ่านข้อมูล ค่าจากสายสัญญาณคอลัมน์จะถูกส่งผ่าน วงจรขยายสัญญาณการตรวจรู้ (Sense Amplifier) เพื่อส่งออกไปยังบัฟเฟอร์ข้อมูล ส่วนในกระบวนการเขียนข้อมูล ตัวขับสัญญาณบิต (Bit driver) จะกระตุ้นสายสัญญาณในแต่ละคอลัมน์เพื่อกำหนดสถานะเป็น 0 หรือ 1 ตามข้อมูลที่ได้รับ

การเพิ่มประสิทธิภาพและการประหยัดพิน (Pin Optimization) เพื่อลดจำนวนพิน (Pins) บนตัวชิป ชิปหน่วยความจำส่วนใหญ่จะใช้การ มัลติเพล็กซ์ที่อยู่ (Address Multiplexing) แทนที่จะใช้สายที่อยู่ทั้งหมด 22 เส้นสำหรับอาเรย์ขนาด 2,048 x 2,048 ชิปจะใช้สายที่อยู่เพียง 11 เส้น (A0–A10) โดยจะส่งที่อยู่แถวเข้าไปก่อน ตามด้วยการส่งที่อยู่คอลัมน์ กระบวนการนี้จะถูกควบคุมด้วยสัญญาณ RAS (Row Address Select) และ CAS (Column Address Select) เพื่อกำหนดจังหวะการทำงาน

นอกจากนี้ บนชิปยังมีพินควบคุมที่สำคัญอื่นๆ ได้แก่
- Write Enable (WE): สำหรับกำหนดการทำงานในโหมดเขียน
- Output Enable (OE): สำหรับกำหนดการทำงานในโหมดอ่าน
- วงจรรีเฟรช (Refresh Circuitry): เฉพาะในชิป DRAM ซึ่งจำเป็นต้องมีการอ่านและเขียนข้อมูลกลับลงไปใหม่ในทุกแถวอย่างสม่ำเสมอ เพื่อป้องกันการสูญหายของประจุไฟฟ้าในเซลล์หน่วยความจำ

### การบรรจุภัณฑ์ชิป (Chip Packaging)
วงจรรวม (Integrated Circuit) จะถูกติดตั้งลงบนบรรจุภัณฑ์ (Package) ที่ประกอบด้วยพิน (Pins) หรือขาสัญญาณ เพื่อใช้สำหรับเชื่อมต่อกับอุปกรณ์ภายนอก

**ตัวอย่างโครงสร้างพินของชิปหน่วยความจำ** เพื่อให้เห็นภาพการทำงาน ชิปหน่วยความจำแต่ละประเภทจะมีจำนวนและหน้าที่ของพินที่แตกต่างกันตามการออกแบบ ดังนี้
1. ชิป EPROM (ตัวอย่างขนาด 8 เมกะบิต) ในชิปที่จัดโครงสร้างแบบ 1M × 8 บิต ซึ่งถือว่าเป็นบรรจุภัณฑ์แบบหนึ่งเวิร์ดต่อหนึ่งชิป (one-word-per-chip package) มักจะใช้บรรจุภัณฑ์มาตรฐานที่มี 32 พิน โดยมีสายสัญญาณที่สำคัญดังนี้
    - สายสัญญาณที่อยู่ (Address lines): สำหรับหน่วยความจำขนาด 1 ล้านเวิร์ด จำเป็นต้องใช้พินที่อยู่จำนวน 20 พิน (A0–A19) เนื่องจาก 2<sup>20</sup> = 1M
    - สายสัญญาณข้อมูล (Data lines): ประกอบด้วย 8 สาย (D0–D7) สำหรับการอ่านข้อมูลออก
    - พินแหล่งจ่ายไฟและกราวด์: ประกอบด้วยพินแรงดันไฟฟ้า (Vcc) และพินกราวด์ (Vss)
    - พินเปิดใช้งานชิป (Chip Enable - CE): เนื่องจากอาจมีชิปหน่วยความจำหลายตัวเชื่อมต่ออยู่บนบัสที่อยู่เดียวกัน พิน CE จึงทำหน้าที่ระบุว่าที่อยู่บนบัสนั้นเป็นของชิปตัวนี้หรือไม่ โดยจะทำงานร่วมกับลอจิกที่เชื่อมต่อกับบิตลำดับสูงของบัสที่อยู่
    - พินแรงดันไฟฟ้าสำหรับการโปรแกรม (Program Voltage - Vpp): ใช้สำหรับป้อนแรงดันไฟฟ้าในระหว่างกระบวนการเขียนข้อมูลหรือการโปรแกรม
2. ชิป DRAM (ตัวอย่างขนาด 16 เมกะบิต) สำหรับการจัดโครงสร้างแบบ 4M × 4 บิต จะมีความแตกต่างจากชิป ROM หลายประการ ดังนี้
   - พินข้อมูลแบบอินพุต/เอาต์พุต (I/O): เนื่องจาก RAM สามารถแก้ไขข้อมูลได้ พินข้อมูลจึงทำหน้าที่เป็นทั้งทางเข้าและทางออกของข้อมูล
   - พินควบคุมการอ่านและเขียน: ได้แก่ WE (Write Enable) และ OE (Output Enable) เพื่อกำหนดว่าจะทำการเขียนหรืออ่านข้อมูล
   - การลดจำนวนพินที่อยู่: เนื่องจาก DRAM ใช้การเข้าถึงแบบแถวและคอลัมน์ร่วมกับการมัลติเพล็กซ์ที่อยู่ จึงใช้พินที่อยู่เพียง 11 พิน เพื่อระบุตำแหน่งรวมทั้งหมด 4 ล้านตำแหน่ง (2<sup>11</sup>×2<sup>11</sup> = 2<sup>22</sup> = 4M)
   - พินควบคุมจังหวะ: ได้แก่ RAS (Row Address Select) และ CAS (Column Address Select) ที่ใช้ควบคุมจังหวะการส่งที่อยู่แถวและคอลัมน์
   - พิน NC (No Connect): เป็นพินที่ไม่มีการเชื่อมต่อ เพื่อให้ตัวบรรจุภัณฑ์มีจำนวนพินเป็นเลขคู่ตามมาตรฐาน
### การจัดโครงสร้างโมดูล (Module Organization)
ในกรณีที่ชิปหน่วยความจำหนึ่งตัวถูกจัดโครงสร้างให้เก็บข้อมูลได้เพียงหนึ่งบิตต่อหนึ่งเวิร์ด จำเป็นต้องใช้จำนวนชิปเท่ากับจำนวนบิตในหนึ่งเวิร์ดเพื่อประกอบกันเป็นโมดูลหน่วยความจำ
 ตัวอย่างเช่น การจัดโครงสร้างโมดูลหน่วยความจำขนาด 256K เวิร์ด โดยที่แต่ละเวิร์ดมีขนาด 8 บิต จะต้องใช้ชิปขนาด 256K × 1 บิต จำนวน 8 ตัว โดยที่สายสัญญาณที่อยู่ (Address lines) ขนาด 18 บิต จะถูกส่งจากแหล่งกำเนิดภายนอกไปยังชิปทั้ง 8 ตัวพร้อมกัน เพื่อให้อินพุต/เอาต์พุตของแต่ละชิปทำหน้าที่แทนข้อมูลแต่ละบิตในเวิร์ดนั้นๆ
 
ในกรณีที่ต้องการหน่วยความจำที่มีความจุสูงขึ้น สามารถทำได้โดยการจัดเรียงชิปในรูปแบบอาเรย์ (Array) ตัวอย่างเช่น การสร้างหน่วยความจำขนาด 1M เวิร์ด (เวิร์ดละ 8 บิต) สามารถทำได้โดยการจัดกลุ่มชิปออกเป็น 4 คอลัมน์ โดยแต่ละคอลัมน์จะมีโครงสร้างโมดูลขนาด 256K × 8 บิต สำหรับหน่วยความจำขนาด 1M เวิร์ดนี้ จะใช้สายสัญญาณที่อยู่ทั้งหมด 20 เส้น โดยที่ 18 เส้นในลำดับล่างจะถูกส่งไปยังทุกโมดูล ส่วน 2 เส้นในลำดับสูง (High-order bits) จะถูกส่งเข้าสู่ตรรกะการเลือกกลุ่ม (Group select logic) เพื่อส่งสัญญาณเลือกชิป (Chip Enable - CE) ไปยังคอลัมน์ของโมดูลที่ต้องการเพียงกลุ่มเดียว

### หน่วยความจำแบบสลับส่วน (Interleaved Memory)
หน่วยความจำหลักประกอบด้วยการรวมกลุ่มของชิปหน่วยความจำ DRAM หลายตัวเข้าด้วยกัน ซึ่งสามารถจัดกลุ่มชิปเหล่านี้เป็นหน่วยที่เรียกว่า "แบงก์หน่วยความจำ" (Memory Bank) โครงสร้างในลักษณะนี้เรียกว่า หน่วยความจำแบบสลับส่วน (Interleaved Memory)

**คุณลักษณะที่สำคัญของหน่วยความจำแบบสลับส่วน ได้แก่**
- การทำงานที่เป็นอิสระ: แต่ละแบงก์มีความสามารถในการรองรับคำขออ่านหรือเขียนข้อมูลได้โดยอิสระต่อกัน
- การเพิ่มประสิทธิภาพการเข้าถึง: ในระบบที่มีแบงก์หน่วยความจำจำนวน K แบงก์ จะสามารถตอบสนองต่อคำขอได้พร้อมกัน K รายการในเวลาเดียวกัน ซึ่งช่วยเพิ่มอัตราการอ่านหรือเขียนข้อมูลได้สูงสุดถึง K เท่า
- ความเร็วในการถ่ายโอนข้อมูลแบบบล็อก: หากมีการจัดเก็บเวิร์ดข้อมูลที่มีที่อยู่ต่อเนื่องกัน (Consecutive words) กระจายไว้ในแบงก์ที่แตกต่างกัน จะช่วยให้การถ่ายโอนข้อมูลขนาดใหญ่หรือแบบบล็อกทำได้รวดเร็วยิ่งขึ้นอย่างมาก

## 5.2 การแก้ไขข้อผิดพลาด (Error Correction)

ระบบหน่วยความจำสารกึ่งตัวนำอาจเกิดข้อผิดพลาดในระหว่างการทำงาน ซึ่งสามารถจำแนกออกได้เป็นสองประเภทหลัก คือ ความล้มเหลวถาวร (Hard Failure) และ ข้อผิดพลาดแบบซอฟต์ (Soft Error)

- ความล้มเหลวถาวร (Hard Failure): เป็นความบกพร่องทางกายภาพของฮาร์ดแวร์อย่างถาวร ส่งผลให้เซลล์หน่วยความจำไม่สามารถจัดเก็บข้อมูลได้อย่างเชื่อถือได้ โดยเซลล์ที่ได้รับผลกระทบอาจค้างอยู่ที่สถานะ 0 หรือ 1 หรือมีการสลับสถานะไปมาอย่างไม่แน่นอน สาเหตุหลักเกิดจากการใช้งานในสภาพแวดล้อมที่รุนแรง ข้อบกพร่องจากกระบวนการผลิต หรือการเสื่อมสภาพตามอายุการใช้งาน
- ข้อผิดพลาดแบบซอฟต์ (Soft Error): เป็นเหตุการณ์สุ่มที่เกิดขึ้นชั่วคราวและไม่ทำลายอุปกรณ์ แต่จะส่งผลให้ข้อมูลในเซลล์หน่วยความจำเปลี่ยนแปลงไป สาเหตุอาจมาจากปัญหาของแหล่งจ่ายไฟ หรือการรบกวนจากอนุภาคแอลฟา (Alpha particles) ซึ่งเกิดจากการสลายตัวของสารกัมมันตรังสีที่พบได้ในวัสดุทั่วไป

**กระบวนการแก้ไขข้อผิดพลาด** เนื่องจากข้อผิดพลาดทั้งสองประเภทเป็นสิ่งที่ไม่พึงปรารถนา ระบบหน่วยความจำหลักสมัยใหม่จึงมักบรรจุตรรกะสำหรับทั้งการตรวจจับและแก้ไขข้อผิดพลาดไว้ด้วย เมื่อมีการเขียนข้อมูลขนาด M บิตลงในหน่วยความจำ ระบบจะคำนวณรหัส (Code) ขนาด K บิตผ่านฟังก์ชัน f และจัดเก็บข้อมูลรวมทั้งหมดเป็น M+K บิต ในขั้นตอนการอ่านข้อมูล ข้อมูล M บิตที่ถูกดึงออกมาจะถูกนำมาคำนวณรหัสใหม่เพื่อเปรียบเทียบกับรหัสเดิมที่จัดเก็บไว้ ผลการเปรียบเทียบอาจเป็นไปได้ 3 กรณี คือ
1. ไม่พบข้อผิดพลาด: ระบบจะส่งข้อมูลออกไปตามปกติ
2. พบข้อผิดพลาดและสามารถแก้ไขได้: ข้อมูลจะถูกส่งเข้าสู่หน่วยแก้ไข (Corrector) เพื่อปรับปรุงให้ถูกต้องก่อนส่งออกไป
3. พบข้อผิดพลาดแต่ไม่สามารถแก้ไขได้: ระบบจะรายงานสภาวะความผิดพลาดนี้ให้ทราบ

**รหัสแฮมมิง (Hamming Code)** รหัสแก้ไขข้อผิดพลาดที่เรียบง่ายที่สุดคือ รหัสแฮมมิง ซึ่งใช้การเพิ่ม **บิตพาริตี (Parity bits)** ลงในส่วนต่างๆ ของข้อมูลในการพิจารณาจำนวนบิตตรวจสอบ (K) สำหรับข้อมูลที่มีจำนวน M บิต จะต้องเป็นไปตามเงื่อนไขทางคณิตศาสตร์คือ 2<sup>K</sup>−1 ≥ M+K ตัวอย่างเช่น ข้อมูลขนาด 8 บิต จะต้องใช้บิตตรวจสอบจำนวน 4 บิต ทำให้คำข้อมูลรวมมีขนาด 12 บิต

ผลลัพธ์จากการเปรียบเทียบบิตตรวจสอบที่ได้จากการอ่านกับบิตที่คำนวณใหม่จะถูกเรียกว่า ซินโดรม (Syndrome word) ซึ่งเป็นรหัสขนาด K บิตที่ใช้ระบุตำแหน่งของบิตที่เกิดข้อผิดพลาดได้อย่างแม่นยำ 

**รหัส SEC และ SEC-DED** รหัสประเภทที่สามารถแก้ไขข้อผิดพลาดได้ 1 บิต เรียกว่า รหัส SEC (Single-Error-Correcting) อย่างไรก็ตาม หน่วยความจำสารกึ่งตัวนำมักนิยมใช้รหัส SEC-DED (Single-Error-Correcting, Double-Error-Detecting) ซึ่งสามารถแก้ไขข้อผิดพลาด 1 บิต และตรวจจับข้อผิดพลาดที่เกิดขึ้นพร้อมกัน 2 บิตได้ การใช้รหัสเหล่านี้ช่วยเพิ่มความเชื่อถือได้ให้กับระบบ แต่ต้องแลกมาด้วยความซับซ้อนและพื้นที่จัดเก็บที่เพิ่มขึ้น (Overhead) ประมาณ 7% ถึง 22% ของความจุหน่วยความจำทั้งหมด

## 5.3 DDR DRAM

ตามที่ได้กล่าวถึงในบทที่ 1 หนึ่งในคอขวดที่สำคัญที่สุดของระบบคอมพิวเตอร์เมื่อใช้งานร่วมกับหน่วยประมวลผลประสิทธิภาพสูง คือช่องทางเชื่อมต่อไปยังหน่วยความจำหลัก (Main Memory) ภายในเครื่อง ช่องทางเชื่อมต่อนี้ถือเป็นเส้นทางที่สำคัญที่สุดในระบบคอมพิวเตอร์ทั้งหมด

หน่วยความจำหลักยังคงมีพื้นฐานมาจากชิป DRAM เหมือนที่ใช้กันมานานหลายทศวรรษ โดยที่ผ่านมายังไม่มีการเปลี่ยนแปลงเชิงสถาปัตยกรรมของ DRAM ที่มีนัยสำคัญนับตั้งแต่ช่วงต้นทศวรรษที่ 1970 ชิป DRAM แบบดั้งเดิมมีข้อจำกัดทั้งในด้านสถาปัตยกรรมภายในและข้อจำกัดของช่องทางเชื่อมต่อกับบัสหน่วยความจำ (Memory Bus) ของหน่วยประมวลผล

แนวทางหนึ่งในการแก้ปัญหาด้านประสิทธิภาพของหน่วยความจำหลัก DRAM คือการเพิ่มระดับหน่วยความจำแคชแบบ SRAM ความเร็วสูง (SRAM Cache) เข้าไประหว่างหน่วยความจำหลัก DRAM กับหน่วยประมวลผล อย่างไรก็ตาม SRAM มีต้นทุนที่สูงกว่า DRAM มาก และการขยายขนาดแคชจนเกินจุดหนึ่งจะเริ่มให้ผลตอบแทนที่ลดน้อยลง (Diminishing Returns)

ในช่วงไม่กี่ปีที่ผ่านมา ได้มีการสำรวจแนวทางการปรับปรุงสถาปัตยกรรม DRAM พื้นฐานหลายรูปแบบ ซึ่งเทคโนโลยีที่ครองตลาดอยู่ในปัจจุบันคือ SDRAM และ DDR-DRAM โดยเราจะศึกษาเทคโนโลยีแต่ละประเภทตามลำดับต่อไป

### Synchronous DRAM

**SDRAM** เป็นหนึ่งในรูปแบบของหน่วยความจำ DRAM ที่มีการใช้งานอย่างแพร่หลายที่สุดในปัจจุบัน โดยมีความแตกต่างที่สำคัญจาก DRAM แบบดั้งเดิมที่เป็นแบบอซิงโครนัส (Asynchronous) คือ SDRAM จะทำการแลกเปลี่ยนข้อมูลกับตัวประมวลผลโดยปรับจังหวะให้สอดคล้องกับสัญญาณนาฬิกาภายนอก (External clock signal) และทำงานด้วยความเร็วสูงสุดของบัสระบบโดยไม่มีสถานะรอคอย (Wait states)

**หลักการทำงานและความแตกต่างจาก DRAM ทั่วไป**
- DRAM แบบดั้งเดิม: เมื่อตัวประมวลผลส่งที่อยู่และสัญญาณควบคุมไปยังหน่วยความจำ ตัวประมวลผลจะต้องรอคอยในช่วงเวลาการเข้าถึง (Access time) เพื่อให้ DRAM ดำเนินกระบวนการภายใน เช่น การกระตุ้นสายสัญญาณแถวและคอลัมน์ และการตรวจรู้ข้อมูล ซึ่งความหน่วงนี้ส่งผลให้ประสิทธิภาพของระบบลดลง
- SDRAM: การเข้าถึงข้อมูลจะเป็นแบบซิงโครนัสภายใต้การควบคุมของสัญญาณนาฬิการะบบ เมื่อตัวประมวลผลส่งคำสั่งและที่อยู่ข้อมูลไปยัง SDRAM ข้อมูลจะถูกส่งกลับมาหลังจากผ่านไปตามจำนวนรอบสัญญาณนาฬิกาที่กำหนดไว้ล่วงหน้า ในระหว่างที่ SDRAM กำลังประมวลผลคำสั่งนั้น ตัวประมวลผล (Master) สามารถไปปฏิบัติงานอื่นได้อย่างปลอดภัย

**คุณสมบัติเด่นของ SDRAM**
1. โหมดเบิสต์ (Burst Mode): ช่วยให้สามารถส่งถ่ายข้อมูลชุดหนึ่งออกมาได้อย่างรวดเร็วหลังจากเข้าถึงบิตแรกสำเร็จ โดยไม่เสียเวลาในการตั้งค่าที่อยู่ (Address setup) และการชาร์จประจุสายสัญญาณใหม่ (Precharge time) โหมดนี้จะมีประสิทธิภาพสูงสุดเมื่อข้อมูลที่ต้องการเข้าถึงวางเรียงลำดับกันอยู่ในแถวเดียวกัน
2. สถาปัตยกรรมแบบหลายแบงก์ (Multiple-bank Internal Architecture): โครงสร้างภายในประกอบด้วยหลายแบงก์ ซึ่งช่วยเพิ่มโอกาสในการทำงานแบบขนาน (Parallelism) ภายในชิปตัวเดียว
3. เรจิสเตอร์โหมด (Mode Register): เป็นคุณสมบัติสำคัญที่ช่วยให้ผู้เขียนโปรแกรมสามารถปรับแต่ง SDRAM ให้เหมาะสมกับความต้องการของระบบได้ เช่น การกำหนดความยาวเบิสต์ (Burst length) หรือจำนวนข้อมูลที่จะส่งเข้าสู่บัสในแต่ละครั้ง และการปรับค่าความหน่วง (Latency) ระหว่างการได้รับคำสั่งอ่านข้อมูลไปจนถึงจุดเริ่มต้นของการถ่ายโอนข้อมูล

SDRAM จะทำงานได้ดีที่สุดเมื่อมีการถ่ายโอนข้อมูลขนาดใหญ่ในลำดับที่ต่อเนื่องกัน เช่น การใช้งานในโปรแกรมประมวลผลคำ (Word processing) ตารางคำนวณ (Spreadsheets) และงานด้านมัลติมีเดีย

### DDR SDRAM

แม้ว่า SDRAM จะช่วยเพิ่มประสิทธิภาพได้มากกว่า RAM แบบอซิงโครนัสอย่างมาก แต่ยังมีข้อจำกัดที่ขัดขวางอัตราการรับส่งข้อมูล (I/O data rate) ดังนั้นจึงได้มีการพัฒนา DDR DRAM (Double-Data-Rate DRAM) ขึ้นโดยสมาคมเทคโนโลยีสารกึ่งตัวนำ JEDEC เพื่อเพิ่มอัตราการส่งข้อมูลให้สูงขึ้นอย่างก้าวกระโดด
**กลไกการเพิ่มความเร็วของ DDR** DDR สามารถบรรลุอัตราการส่งข้อมูลที่สูงขึ้นผ่าน 3 วิธีการหลัก ดังนี้
1. การส่งข้อมูลแบบอัตราสองเท่า: การรับส่งข้อมูลจะถูกปรับจังหวะให้ตรงกับทั้ง ขอบขาขึ้น (Rising edge) และ ขอบขาลง (Falling edge) ของสัญญาณนาฬิกา แทนที่จะใช้เพียงขอบขาขึ้นเพียงอย่างเดียวเหมือน SDRAM ปกติ ซึ่งช่วยให้ส่งข้อมูลได้เป็นสองเท่าในหนึ่งรอบสัญญาณนาฬิกา
2. การเพิ่มความเร็วสัญญาณนาฬิกา: มีการใช้ความเร็วสัญญาณนาฬิกาบนบัสที่สูงขึ้น
3. ระบบบัฟเฟอร์ (Buffering Scheme): หรือที่เรียกว่า บัฟเฟอร์การดึงข้อมูลล่วงหน้า (Prefetch Buffer) ซึ่งเป็นแคชหน่วยความจำที่ติดตั้งอยู่บนชิป SDRAM

**บัฟเฟอร์การดึงข้อมูลล่วงหน้า (Prefetch Buffer)** เนื่องจากแกนหลัก (Core) ของ DRAM ทำงานช้ากว่าส่วนอินเทอร์เฟซภายนอกมาก ระบบจึงต้องใช้การเข้าถึงข้อมูลแบบขนานแล้วนำมาเปลี่ยนเป็นข้อมูลแบบอนุกรมผ่านตัวมัลติเพล็กซ์ (MUX) ขนาดของบัฟเฟอร์นี้จะเป็นตัวกำหนดจำนวนเวิร์ดที่ถูกดึงออกมาในแต่ละครั้ง ตัวอย่างเช่น DDR ที่มีบัฟเฟอร์ขนาด 2 บิต จะทำการดึงข้อมูลล่วงหน้า 2 เวิร์ดในทุกครั้งที่มีการอ่านหรือเขียน ทำให้ส่วนอินเทอร์เฟซทำงานได้เร็วกว่าแกนหลักของ SDRAM ถึงสองเท่า

**วิวัฒนาการของเทคโนโลยี DDR** JEDEC ได้กำหนดมาตรฐาน DDR ออกเป็นหลายชั่วอายุ (Generations) โดยมีการพัฒนาที่สำคัญดังนี้
- DDR1: ใช้บัฟเฟอร์การดึงข้อมูลล่วงหน้าขนาด 2 บิต
- DDR2: เพิ่มขนาดบัฟเฟอร์เป็น 4 บิต ช่วยเพิ่มอัตราการส่งข้อมูลเป็น 4 เท่า
- DDR3: เพิ่มขนาดบัฟเฟอร์เป็น 8 บิต ช่วยเพิ่มประสิทธิภาพได้ถึง 8 เท่า
- DDR4: แทนที่จะเพิ่มขนาดบัฟเฟอร์เป็น 16 บิต (ซึ่งจะส่งผลต่อความยาวเบิสต์ขั้นต่ำที่ยาวเกินไป) ผู้พัฒนาได้นำแนวคิด กลุ่มแบงก์ (Bank Groups) มาใช้ โดยอนุญาตให้มีการดึงข้อมูลล่วงหน้าขนาด 8 บิต สองชุดทำงานขนานกันในกลุ่มแบงก์ที่แยกจากกัน ทำให้สามารถเพิ่มประสิทธิภาพได้โดยไม่ต้องเปลี่ยนขนาดบัฟเฟอร์พื้นฐาน

นอกจากนี้ DDR ในแต่ละรุ่นยังมีการลดระดับแรงดันไฟฟ้าลง (จาก 2.5V ใน DDR1 เหลือเพียง 1.2V ใน DDR4) เพื่อลดการใช้พลังงานในขณะที่เพิ่มความเร็วในการรับส่งข้อมูลบนบัสให้สูงขึ้น

## 5.4 หน่วยความจำแฟลช (Flash Memory)
หน่วยความจำแฟลช เป็นหน่วยความจำสารกึ่งตัวนำประเภทไม่ระเหย (Nonvolatile) ซึ่งนิยมใช้งานอย่างแพร่หลายทั้งในฐานะหน่วยความจำภายในและหน่วยความจำภายนอก โดยเทคโนโลยีนี้ถูกนำเสนอเป็นครั้งแรกในช่วงกลางทศวรรษที่ 1980 ซึ่งมีคุณสมบัติและต้นทุนอยู่กึ่งกลางระหว่าง EPROM และ EEPROM
**คุณลักษณะเด่นและการลบข้อมูล** หน่วยความจำแฟลชใช้เทคโนโลยีการลบข้อมูลด้วยสัญญาณไฟฟ้าเช่นเดียวกับ EEPROM แต่มีความโดดเด่นที่ความเร็วในการลบข้อมูล ซึ่งสามารถลบข้อมูลทั้งชิปได้ภายในเวลาเพียงไม่กี่วินาที อย่างไรก็ตาม แฟลชไม่รองรับการลบข้อมูลในระดับไบต์ (Byte-level erasure) แต่จะทำการลบข้อมูลเป็นส่วนๆ หรือเป็นบล็อก (Blocks) ซึ่งเป็นที่มาของชื่อ "แฟลช" เนื่องจากความสามารถในการลบข้อมูลส่วนประกอบของชิปได้ในครั้งเดียว นอกจากนี้ แฟลชยังใช้ทรานซิสเตอร์เพียงหนึ่งตัวต่อหนึ่งบิต ทำให้มีความหนาแน่นของข้อมูลสูงใกล้เคียงกับ EPROM

### กลไกการทำงาน (Operation) 
การทำงานของเซลล์หน่วยความจำแฟลชอาศัยโครงสร้างทรานซิสเตอร์ที่เพิ่มเกตที่สองซึ่งเรียกว่า "เกตลอย" (Floating Gate)
- สถานะทางตรรกะ: ในสถานะเริ่มต้น เกตลอยจะไม่มีประจุไฟฟ้า ซึ่งแทนค่าเลขฐานสองเป็น 1
- การเขียนข้อมูล: เมื่อมีการป้อนแรงดันไฟฟ้าสูงข้ามชั้นออกไซด์ อิเล็กตรอนจะถูกกักเก็บไว้ในเกตลอยแม้จะไม่มีกระแสไฟฟ้าเลี้ยง ซึ่งในสภาวะนี้จะแทนค่าเลขฐานสองเป็น 0
- การอ่านข้อมูล: ระบบจะใช้วงจรภายนอกตรวจสอบว่าทรานซิสเตอร์ทำงานหรือไม่เพื่อระบุสถานะของเซลล์

### ประเภทของหน่วยความจำแฟลช 
เทคโนโลยีแฟลชแบ่งออกเป็นสองสถาปัตยกรรมหลักตามลักษณะโครงสร้างทางตรรกะ ดังนี้
1. NOR Flash
   - โครงสร้าง: เซลล์หน่วยความจำถูกเชื่อมต่อแบบขนานกับสายสัญญาณบิต (Bit lines) คล้ายกับโครงสร้างของตรรกะเกตแบบ NOR
   - คุณสมบัติ: รองรับการเข้าถึงข้อมูลแบบสุ่มความเร็วสูง (High-speed random access) สามารถอ่านและเขียนข้อมูลในตำแหน่งที่ระบุหรือดึงข้อมูลระดับไบต์ได้โดยตรง
   - การใช้งาน: เหมาะสำหรับเป็นหน่วยความจำภายในในระบบฝังตัว (Embedded Systems) เพื่อใช้ในการเก็บและรันรหัสโปรแกรม (Code execution)
2. NAND Flash
   - โครงสร้าง: จัดวางทรานซิสเตอร์แบบอนุกรม คล้ายกับตรรกะเกตแบบ NAND
   - คุณสมบัติ: มีความหนาแน่นของบิตสูงกว่าและมีความเร็วในการเขียนข้อมูลที่เหนือกว่าแบบ NOR แต่ไม่รองรับบัสที่อยู่ภายนอกสำหรับการเข้าถึงแบบสุ่ม จึงต้องอาศัยการเข้าถึงข้อมูลเป็นบล็อก (Page access)
   - การใช้งาน: เหมาะสำหรับใช้เป็นหน่วยความจำภายนอก เช่น USB Flash Drives, การ์ดหน่วยความจำ และ SSD (Solid-State Disks)
  
## 5.5 เทคโนโลยีหน่วยความจำโซลิดสเตตแบบไม่ระเหยรุ่นใหม่
ในปัจจุบัน การพัฒนาหน่วยความจำสารกึ่งตัวนำแบบเดิม (เช่น SRAM, DRAM และ Flash) ให้มีขนาดเล็กลง (Scaling) มีประสิทธิภาพสูงขึ้น และมีราคาถูกลงนั้นเริ่มทำได้ยากขึ้นตามข้อจำกัดทางกายภาพ จึงได้มีการพัฒนาเทคโนโลยีหน่วยความจำแบบไม่ระเหย (Nonvolatile) รุ่นใหม่ขึ้นมาเพื่อก้าวข้ามขีดจำกัดดังกล่าว โดยเทคโนโลยีที่สำคัญและมีแนวโน้มจะเข้ามามีบทบาทในลำดับชั้นของหน่วยความจำ (Memory Hierarchy) มีดังนี้

### STT-RAM (Spin-Transfer Torque RAM) 
STT-RAM เป็นหน่วยความจำแม่เหล็ก (MRAM) ประเภทใหม่ที่มีคุณสมบัติไม่ระเหย มีความเร็วในการเขียนและอ่านสูงมาก (ประมาณ 6–10 นาโนวินาที) มีความทนทานต่อการโปรแกรมสูง และไม่ใช้พลังงานในสภาวะสแตนด์บาย
- หลักการทำงาน: ใช้โครงสร้างที่เรียกว่า Magnetic Tunneling Junction (MTJ) ซึ่งประกอบด้วยชั้นแม่เหล็กสองชั้นประกบสารไดอิเล็กทริกบางๆ โดยสถานะตรรกะจะขึ้นอยู่กับทิศทางการวางตัวของสนามแม่เหล็กในชั้นอิสระ (Free layer) ว่าขนานหรือสวนทางกับชั้นอ้างอิง (Reference layer)
- การประยุกต์ใช้: STT-RAM เป็นตัวเลือกที่ดีเยี่ยมสำหรับใช้เป็นหน่วยความจำแคช (Cache) หรือหน่วยความจำหลัก (Main Memory)

### PCRAM (Phase-Change RAM)
เทคโนโลยีนี้ใช้สารผสมประเภทแชลโคจิไนด์ (Chalcogenide alloy) ซึ่งเป็นวัสดุชนิดเดียวกับที่ใช้ในแผ่น CD และ DVD
- หลักการทำงาน: อาศัยความแตกต่างของค่าความต้านทานระหว่างสถานะอสัณฐาน (Amorphous) ที่มีความต้านทานสูง และสถานะผลึก (Crystalline) ที่มีความต้านทานต่ำ โดยการเปลี่ยนสถานะจะใช้พัลส์ไฟฟ้าเพื่อควบคุมอุณหภูมิของวัสดุ
- การประยุกต์ใช้: PCRAM มีความเหมาะสมในการนำมาใช้ทดแทนหรือเสริมการทำงานของ DRAM ในส่วนของหน่วยความจำหลัก

### ReRAM (Resistive RAM)
ReRAM หรือ RRAM ทำงานโดยการสร้างความต้านทานภายในวัสดุแทนการเก็บประจุไฟฟ้าโดยตรง
- หลักการทำงาน: เมื่อป้อนกระแสไฟฟ้าเข้าไปจะทำให้ค่าความต้านทานของวัสดุเปลี่ยนไป ซึ่งสามารถวัดค่าเพื่อระบุสถานะเป็น 0 หรือ 1 ได้
- คุณสมบัติเด่น: ใช้แรงดันไฟฟ้าต่ำ มีความทนทานสูงกว่าหน่วยความจำแฟลชอย่างมาก และในทางทฤษฎีจะมีขนาดเซลล์ที่เล็กมาก
- การประยุกต์ใช้: เป็นตัวเลือกที่เหมาะสมสำหรับทั้งหน่วยความจำรอง (Secondary Storage) และหน่วยความจำหลัก
