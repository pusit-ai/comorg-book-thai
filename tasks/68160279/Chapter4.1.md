# 4.1 Computer Memory System Overview

ระบบหน่วยความจำคอมพิวเตอร์มีความหลากหลายสูง ทั้งในด้านเทคโนโลยี การจัดการ ประสิทธิภาพ และราคา โดยไม่มีเทคโนโลยีเพียงหนึ่งเดียวที่สามารถตอบโจทย์ทุกความต้องการของระบบได้

## ลักษณะสำคัญของหน่วยความจำ (Key Characteristics)
ในการจัดการระบบหน่วยความจำ เราสามารถจำแนกตามลักษณะเด่นได้ดังนี้:
* **ตำแหน่ง (Location):** ภายใน (Internal) เช่น Registers, Cache, Main Memory และ ภายนอก (External) เช่น Disk, Tape
* **ความจุ (Capacity):** วัดเป็น Byte หรือ Word โดยความยาว Word ทั่วไปคือ 8, 16, และ 32 บิต
* **หน่วยการถ่ายโอน (Unit of Transfer):** จำนวนบิตที่อ่านหรือเขียนในครั้งเดียว (Word หรือ Block)
* **วิธีการเข้าถึง (Access Method):** * Sequential (เรียงลำดับ)
    * Direct (เข้าถึงโดยตรงในพื้นที่ใกล้เคียง)
    * Random (สุ่มเข้าถึงตำแหน่งใดก็ได้ด้วยเวลาคงที่)
    * Associative (ค้นหาจากเนื้อหาข้อมูล)
* **ประสิทธิภาพ (Performance):** วัดจาก Access time (เวลาเข้าถึง), Cycle time (รอบเวลา), และ Transfer rate (อัตราการส่งข้อมูล)

## ลำดับชั้นของหน่วยความจำ (The Memory Hierarchy)
เนื่องจากมีข้อจำกัดด้านราคาและความเร็ว ผู้ออกแบบจึงใช้โครงสร้างแบบลำดับชั้น (Hierarchy) เพื่อให้ได้ความจุสูงในราคาที่เหมาะสม:
1. **Inboard Memory:** Registers, Cache, Main Memory (เร็วที่สุด ความจุน้อยที่สุด)
2. **Outboard Storage:** Magnetic Disk, CD-ROM, DVD, Blu-Ray
3. **Off-line Storage:** Magnetic Tape (ช้าที่สุด ความจุมากที่สุด)

**กฎของลำดับชั้น:** ยิ่งลงไปชั้นล่าง ราคาต่อบิตจะถูกลง ความจุจะเพิ่มขึ้น แต่เวลาในการเข้าถึงจะช้าลง และความถี่ในการเข้าถึงจาก Processor จะลดลง
