## 2. Cache Memory คืออะไร
แคชคือหน่วยความจำขนาดเล็กที่มีความเร็วสูงมาก ทำหน้าที่เก็บข้อมูลจาก RAM ที่ CPU เรียกใช้งานบ่อย

* **กระบวนการทำงาน:** เมื่อ CPU ต้องการข้อมูล จะตรวจสอบใน Cache ก่อนเสมอ
* **Cache Hit:** พบข้อมูลใน Cache และส่งให้ CPU ได้ทันที
* **Cache Miss:** ไม่พบข้อมูลใน Cache ระบบต้องดึงจาก RAM มาเก็บไว้ใน Cache แทน
* **หลักการ Locality of Reference:** โปรแกรมมักจะเรียกใช้ข้อมูลเดิมซ้ำๆ หรือข้อมูลที่อยู่ใกล้เคียงกัน เช่น การทำงานใน Loop หรือ Array
