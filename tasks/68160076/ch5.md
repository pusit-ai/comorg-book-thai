บทที่ 5: หน่วยความจำภายใน (Internal Memory)
==============================================

5.1 หน่วยความจำหลักแบบสารกึ่งตัวนำ (Semiconductor Main Memory)
----------------------------------------------------------------

โครงสร้างพื้นฐาน (Organization)
---------------------------------
องค์ประกอบพื้นฐานของหน่วยความจำสารกึ่งตัวนำคือ เซลล์หน่วยความจำ (memory cell) ซึ่งมีคุณสมบัติร่วมกันดังนี้:
- มีสองสถานะที่เสถียร แทนค่าไบนารี 1 และ 0
- สามารถเขียนข้อมูลเพื่อกำหนดสถานะได้
- สามารถอ่านเพื่อตรวจสอบสถานะได้

เซลล์หน่วยความจำมีขั้วงาน 3 ขั้ว ได้แก่:
- Select  — เลือกเซลล์สำหรับอ่านหรือเขียน
- Control — ระบุว่าจะอ่านหรือเขียน
- Data    — รับหรือส่งข้อมูล


DRAM และ SRAM
--------------
หน่วยความจำ RAM แบ่งออกเป็น 2 ประเภทหลัก:

DRAM (Dynamic RAM)
- เก็บข้อมูลในรูปประจุไฟฟ้าในตัวเก็บประจุ (capacitor)
- ต้องทำการรีเฟรช (refresh) เป็นระยะเพราะประจุรั่วไหลตามธรรมชาติ
- มีความหนาแน่นสูง ราคาถูก เหมาะกับหน่วยความจำหลัก (main memory)

SRAM (Static RAM)
- เก็บข้อมูลด้วยวงจร flip-flop แบบดิจิทัล
- ไม่ต้องรีเฟรช ทำงานได้ตราบที่มีไฟเลี้ยง
- เร็วกว่าแต่แพงกว่าและมีความหนาแน่นต่ำกว่า เหมาะกับแคช (cache memory)

เปรียบเทียบ DRAM กับ SRAM:
คุณสมบัติ     | DRAM       | SRAM
ความเร็ว      | ช้ากว่า    | เร็วกว่า
ราคา          | ถูกกว่า    | แพงกว่า
ความหนาแน่น   | สูง        | ต่ำ
ต้องรีเฟรช    | ใช่        | ไม่ใช่
การใช้งาน     | Main Memory| Cache

ทั้ง DRAM และ SRAM เป็นหน่วยความจำระเหย (volatile) คือข้อมูลจะหายเมื่อไฟดับ


ประเภทของ ROM
--------------
ROM (Read-Only Memory) เก็บข้อมูลถาวร ไม่ระเหย (nonvolatile) แต่ไม่สามารถเขียนใหม่ได้ตามปกติ มีหลายประเภท:

ประเภท   | การลบ              | การเขียน
ROM      | ไม่ได้             | ใช้ mask ในการผลิต
PROM     | ไม่ได้             | เขียนได้ครั้งเดียวด้วยไฟฟ้า
EPROM    | แสง UV (ทั้งชิป)   | เขียนด้วยไฟฟ้า
EEPROM   | ไฟฟ้า (ระดับ byte) | เขียนด้วยไฟฟ้า
Flash    | ไฟฟ้า (ระดับ block)| เขียนด้วยไฟฟ้า


Chip Logic และ Module Organization
------------------------------------
- ชิป DRAM ขนาด 16 Mbit มักอ่าน/เขียนครั้งละ 4 บิต โดยใช้ array 2048x2048
- ใช้ address multiplexing เพื่อลดจำนวนขา โดยส่ง row address และ column address แยกกัน พร้อมสัญญาณ RAS และ CAS
- หากต้องการความจุมากขึ้น จะนำหลายชิปมาต่อกันเป็น module

Interleaved Memory
-------------------
การแบ่งหน่วยความจำเป็นหลาย bank ที่ทำงานได้อิสระพร้อมกัน ระบบที่มี K bank สามารถรับ K คำร้องขอพร้อมกัน เพิ่มอัตราการถ่ายโอนข้อมูลได้ K เท่า


5.2 การแก้ไขข้อผิดพลาด (Error Correction)
-------------------------------------------

ข้อผิดพลาดในหน่วยความจำมี 2 ประเภท:
- Hard failure — ความเสียหายถาวร เซลล์ค้างที่ 0 หรือ 1
- Soft error   — เกิดแบบสุ่ม ไม่ถาวร เช่นจากปัญหาไฟหรืออนุภาคแอลฟา

Hamming Code
-------------
เป็น error-correcting code ที่ง่ายที่สุด สำหรับข้อมูล M บิต ต้องการบิตตรวจสอบ K บิต โดยที่:
    2^K - 1 >= M + K

สำหรับข้อมูล 8 บิต ต้องการ check bit 4 บิต รวมเป็น 12 บิต

ตำแหน่ง check bit คือตำแหน่งที่เป็นเลขยกกำลังของ 2 (1, 2, 4, 8) และคำนวณดังนี้:
    C1 = D1 XOR D2 XOR D4 XOR D5 XOR D7
    C2 = D1 XOR D3 XOR D4 XOR D6 XOR D7
    C4 = D2 XOR D3 XOR D4 XOR D8
    C8 = D5 XOR D6 XOR D7 XOR D8

เมื่ออ่านข้อมูลออกมา จะคำนวณ syndrome word โดย XOR check bit ใหม่กับเก่า:
- syndrome = 0000                  -> ไม่มีข้อผิดพลาด
- syndrome มีบิตเดียวเป็น 1       -> check bit เสีย ไม่ต้องแก้ไขข้อมูล
- syndrome มีหลายบิตเป็น 1        -> ค่าตัวเลขของ syndrome คือตำแหน่ง data bit ที่ผิดพลาด

นอกจากนี้ยังมี SEC-DED (Single-Error Correcting, Double-Error Detecting)
ซึ่งใช้ check bit เพิ่มอีก 1 บิต เพื่อตรวจจับข้อผิดพลาด 2 บิตพร้อมกันได้

จำนวน check bit ที่ต้องการตามขนาด data word:
Data Bits | SEC (check bits) | SEC-DED (check bits)
8         | 4                | 5
16        | 5                | 6
32        | 6                | 7
64        | 7                | 8
128       | 8                | 9
256       | 9                | 10


5.3 DDR DRAM
-------------

Synchronous DRAM (SDRAM)
-------------------------
SDRAM ทำงานซิงค์กับสัญญาณนาฬิกาภายนอก ต่างจาก DRAM แบบดั้งเดิมที่เป็น asynchronous
ทำให้ไม่ต้องใส่ wait state และมีประสิทธิภาพสูงขึ้น

คุณสมบัติสำคัญ:
- Burst mode    — ส่งข้อมูลต่อเนื่องหลังจาก access แรกโดยไม่ต้องใส่ address ใหม่
- Mode register — กำหนด burst length และ CAS latency ได้

DDR SDRAM (Double Data Rate)
-----------------------------
DDR เพิ่มอัตราการส่งข้อมูลด้วย 3 วิธี:
1. ส่งข้อมูลทั้งขาขึ้นและขาลงของสัญญาณนาฬิกา (double data rate)
2. ใช้ clock rate ที่สูงขึ้น
3. ใช้ prefetch buffer

รุ่น  | Prefetch Buffer  | แรงดัน (V) | อัตราส่งข้อมูล (Mbps)
DDR1 | 2 บิต            | 2.5        | 200-400
DDR2 | 4 บิต            | 1.8        | 400-1066
DDR3 | 8 บิต            | 1.5        | 800-2133
DDR4 | 8 บิต+bank group | 1.2        | 2133-4266

DDR4 แนะนำแนวคิด bank group แทนการเพิ่ม prefetch buffer
เพื่อหลีกเลี่ยงข้อเสียของ burst length ที่ยาวเกินไป


5.4 Flash Memory
-----------------

Flash memory เป็นหน่วยความจำถาวร (nonvolatile) กึ่งกลางระหว่าง EPROM และ EEPROM

หลักการทำงาน
-------------
ใช้ floating gate transistor:
- สถานะ 1: ไม่มีอิเล็กตรอนบน floating gate ทรานซิสเตอร์ทำงานปกติ
- สถานะ 0: มีอิเล็กตรอนติดค้างบน floating gate ทรานซิสเตอร์ถูกบล็อก
- การลบข้อมูล: ใส่แรงดันย้อนกลับเพื่อดึงอิเล็กตรอนออก

NOR vs NAND Flash
------------------
คุณสมบัติ       | NOR Flash             | NAND Flash
การเชื่อมต่อ    | ขนานกับ bit line      | อนุกรมกัน (16-32 ตัว)
การเข้าถึง      | Random access (byte)  | Block access (page)
ความเร็วอ่าน    | สูง                   | ปานกลาง
ความเร็วเขียน   | ต่ำ                   | สูง
ความหนาแน่น     | ต่ำกว่า               | สูงกว่า
การใช้งาน       | Embedded (program)    | USB, SSD, memory cards


5.5 เทคโนโลยีหน่วยความจำแบบไม่ระเหยรุ่นใหม่
----------------------------------------------

มี 3 เทคโนโลยีที่มีแนวโน้มสำคัญ:

STT-RAM (Spin-Transfer Torque RAM)
-----------------------------------
- ใช้ magnetic tunneling junction (MTJ) เก็บข้อมูลด้วยทิศทางของสนามแม่เหล็ก
- ข้อดี: ไม่ระเหย, เขียน/อ่านเร็ว (<10 ns), ทนทานสูง (>10^15 รอบ), ไม่ใช้ไฟในโหมด standby
- เหมาะแทนที่ cache หรือ main memory

PCRAM (Phase-Change RAM)
------------------------
- ใช้วัสดุ chalcogenide alloy ที่เปลี่ยนสถานะระหว่าง:
  amorphous (ความต้านทานสูง = 0) และ crystalline (ความต้านทานต่ำ = 1)
- เป็นเทคโนโลยีที่เติบโตมากที่สุดในกลุ่มนี้
- เหมาะแทนที่หรือเสริม DRAM สำหรับ main memory

ReRAM (Resistive RAM)
---------------------
- เก็บข้อมูลด้วยการเปลี่ยนความต้านทานของวัสดุด้วยกระแสไฟฟ้า
- แรงดันต่ำ ทนทานสูงกว่า flash มาก เซลล์มีขนาดเล็กกว่า
- เหมาะแทนที่ secondary storage และ main memory

ลำดับชั้นของหน่วยความจำที่คาดการณ์ในอนาคต:
ประสิทธิภาพสูง / ต้นทุนสูง
        ^
      SRAM (Cache)
      STT-RAM
      PCRAM
      ReRAM
      DRAM (Main Memory)
      NAND Flash
      Hard Disk
        v
ความจุสูง / ต้นทุนต่ำ


สรุป:
------
บทนี้ครอบคลุมวิวัฒนาการของหน่วยความจำหลัก ตั้งแต่ SRAM/DRAM พื้นฐาน ไปจนถึง ROM หลายประเภท
การแก้ไขข้อผิดพลาดด้วย Hamming Code การปรับปรุงประสิทธิภาพด้วย DDR DRAM
และเทคโนโลยีใหม่อย่าง Flash Memory และ Nonvolatile RAM รุ่นต่าง ๆ
