## 4.1 ภาพรวมระบบหน่วยความจำของคอมพิวเตอร์ (Computer Memory System Overview)

### ลักษณะของระบบหน่วยความจำ (Characteristics of Memory Systems)

ระบบหน่วยความจำของคอมพิวเตอร์มีหลายประเภทและมีความแตกต่างกันในด้านโครงสร้าง ประสิทธิภาพ และต้นทุน โดยสามารถแบ่งตามลักษณะสำคัญ เช่น

* ตำแหน่งของหน่วยความจำ (ภายใน / ภายนอก)
* ความจุ (Capacity)
* หน่วยของการถ่ายโอนข้อมูล (Unit of Transfer)
* วิธีการเข้าถึงข้อมูล (Access Method)
* ประสิทธิภาพ (Performance)
* ประเภททางกายภาพ (Physical Type)
* ลักษณะทางกายภาพ (Physical Characteristics)
* การจัดโครงสร้าง (Organization)

### ลำดับชั้นของหน่วยความจำ (Memory Hierarchy)

เนื่องจากไม่มีหน่วยความจำชนิดใดที่ดีที่สุดในทุกด้าน ระบบคอมพิวเตอร์จึงใช้ “ลำดับชั้นของหน่วยความจำ” (Memory Hierarchy) เพื่อให้เกิดความสมดุลระหว่าง

* ความเร็ว (Speed)
* ความจุ (Capacity)
* ต้นทุน (Cost)

ลักษณะของลำดับชั้น:

* ยิ่งเร็ว → ยิ่งแพง → ความจุน้อย
* ยิ่งช้า → ยิ่งถูก → ความจุมาก

ตัวอย่างลำดับ:

* Register (เร็วที่สุด)
* Cache
* Main Memory (RAM)
* Secondary Storage (เช่น HDD, SSD)

หลักการสำคัญคือ **Locality of Reference**
โปรแกรมมักจะเรียกใช้ข้อมูลเดิมซ้ำ ๆ ในช่วงเวลาหนึ่ง ทำให้สามารถเก็บข้อมูลไว้ในหน่วยความจำที่เร็วกว่าได้
