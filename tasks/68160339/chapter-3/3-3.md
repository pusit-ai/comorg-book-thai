## 3.3 โครงสร้างการเชื่อมต่อระหว่างกัน

----

คอมพิวเตอร์ประกอบด้วยชุดของส่วนประกอบหรือโมดูลพื้นฐาน 3 ประเภท (ตัวประมวลผล, หน่วยความจำ, อินพุต/เอาต์พุต) ที่สื่อสารระหว่างกัน ในทางปฏิบัติ คอมพิวเตอร์คือเครือข่ายของโมดูลพื้นฐาน ดังนั้นจึงต้องมีเส้นทางสำหรับการเชื่อมต่อโมดูลเหล่านี้เข้าด้วยกัน

กลุ่มของเส้นทางที่เชื่อมต่อโมดูลต่าง ๆ เข้าด้วยกันนี้เรียกว่า โครงสร้างการเชื่อมต่อระหว่างกัน (Interconnection Structure) การออกแบบโครงสร้างนี้จะขึ้นอยู่กับการแลกเปลี่ยนข้อมูลที่ต้องเกิดขึ้นระหว่างโมดูลเหล่านั้น

* หน่วยความจำ (Memory) : โดยปกติแล้ว โมดูลหน่วยความจำจะประกอบด้วย เวิร์ด (Word) จำนวน $N$ เวิร์ดที่มีความยาวเท่ากัน แต่ละเวิร์ดจะถูกกำหนดด้วย เลขที่อยู่ (Address) ที่เป็นตัวเลขเฉพาะตัว ($0, 1, \dots, N-1$) ข้อมูลในแต่ละเวิร์ดสามารถถูกอ่านออกมาจากหรือเขียนลงไปในหน่วยความจำได้ โดยลักษณะของการทำงานนั้นจะถูกระบุด้วยสัญญาณควบคุมการอ่านและเขียน (read and write control signals) ส่วนตำแหน่งสำหรับการปฏิบัติการนั้นจะถูกระบุโดยเลขที่อยู่
* โมดูลอินพุต/เอาต์พุต (I/O module) : หากมองจากมุมมองภายใน (ของระบบคอมพิวเตอร์) โมดูล I/O มีหน้าที่การทำงานคล้ายคลึงกับหน่วยความจำ โดยมีการปฏิบัติการสองอย่างคือ การอ่าน (read) และ การเขียน (write) นอกจากนี้ โมดูล I/O หนึ่งโมดูลอาจควบคุมอุปกรณ์ภายนอกได้มากกว่าหนึ่งเครื่อง เราสามารถเรียกส่วนเชื่อมต่อ (interface) ของอุปกรณ์ภายนอกแต่ละตัวว่า พอร์ต (port) และกำหนดเลขที่อยู่เฉพาะตัว (unique address) ให้แก่แต่ละพอร์ตได้ (เช่น $0, 1, \dots, M-1$) ยิ่งไปกว่านั้น ยังมีเส้นทางข้อมูลภายนอกสำหรับการนำเข้าและส่งออกข้อมูลกับอุปกรณ์ภายนอก และประการสุดท้าย โมดูล I/O อาจสามารถส่ง สัญญาณขัดจังหวะ (interrupt signals) ไปยังตัวประมวลผลได้
* ตัวประมวลผล (Processor) : ตัวประมวลผลทำหน้าที่อ่านคำสั่งและข้อมูล เขียนข้อมูลออกไปหลังจากประมวลผลเสร็จสิ้น และใช้สัญญาณควบคุมในการควบคุมการทำงานโดยรวมของระบบ นอกจากนี้ยังทำหน้าที่รับสัญญาณขัดจังหวะอีกด้วย

รายการที่กล่าวมาข้างต้นได้กำหนดข้อมูลที่ต้องใช้ในการแลกเปลี่ยน โครงสร้างการเชื่อมต่อระหว่างกัน (Interconnection structure) จะต้องรองรับประเภทของการถ่ายโอนข้อมูลดังต่อไปนี้
* หน่วยความจำไปยังตัวประมวลผล (Memory to processor): ตัวประมวลผลอ่านคำสั่งหรือหน่วยข้อมูลจากหน่วยความจำ
* ตัวประมวลผลไปยังหน่วยความจำ (Processor to memory): ตัวประมวลผลเขียนหน่วยข้อมูลลงในหน่วยความจำ
* I/O ไปยังตัวประมวลผล (I/O to processor): ตัวประมวลผลอ่านข้อมูลจากอุปกรณ์ I/O ผ่านทางโมดูล I/O
* ตัวประมวลผลไปยัง I/O (Processor to I/O): ตัวประมวลผลส่งข้อมูลไปยังอุปกรณ์ I/O
* I/O ไปยังหรือจากหน่วยความจำ (I/O to or from memory): สำหรับสองกรณีนี้ โมดูล I/O จะได้รับอนุญาตให้แลกเปลี่ยนข้อมูลกับหน่วยความจำโดยตรง โดยไม่ต้องผ่านตัวประมวลผล ซึ่งใช้การเข้าถึงหน่วยความจำโดยตรง (Direct Memory Access หรือ DMA)

"ตลอดหลายปีที่ผ่านมา ได้มีการทดลองใช้โครงสร้างการเชื่อมต่อระหว่างกันจำนวนมาก แต่รูปแบบที่แพร่หลายที่สุดจนถึงปัจจุบันคือ (1) ระบบบัส (bus) และโครงสร้างแบบหลายบัส (multiple-bus structures) ต่างๆ และ (2) โครงสร้างการเชื่อมต่อแบบจุดต่อจุด (point-to-point interconnection structures) ที่มีการถ่ายโอนข้อมูลแบบแพ็กเก็ต (packetized data transfer) โดยเนื้อหาส่วนที่เหลือของบทนี้ เราจะมุ่งเน้นไปที่การอภิปรายเกี่ยวกับโครงสร้างเหล่านี้
