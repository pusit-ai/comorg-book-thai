## 4.1 ภาพรวมระบบหน่วยความจำคอมพิวเตอร์ (Computer Memory System Overview)

### คุณลักษณะของระบบหน่วยความจำ (Characteristics of Memory Systems)
ระบบหน่วยความจำสามารถจำแนกตามคุณลักษณะสำคัญได้ดังนี้:

| คุณลักษณะ | รายละเอียด |
| :--- | :--- |
| **สถานที่ (Location)** | - **ภายใน (Internal):** Registers, Cache, Main Memory (RAM)<br>- **ภายนอก (External):** HDD, SSD, Tape |
| **ความจุ (Capacity)** | ระบุเป็น Byte หรือ Word |
| **หน่วยของการถ่ายโอน** | - **ภายใน:** ตามจำนวนสายข้อมูล (Bus Width)<br>- **ภายนอก:** ถ่ายโอนเป็น "บล็อก" (Blocks) |
| **วิธีการเข้าถึง (Access)** | - **Sequential:** ตามลำดับ (Tape)<br>- **Direct:** เข้าถึงตำแหน่งแล้วหาต่อ (Disk)<br>- **Random:** เข้าถึงได้ทุกที่ในเวลาเท่ากัน (RAM)<br>- **Associative:** ค้นจากเนื้อหาแทนที่อยู่ |

### ลำดับชั้นของหน่วยความจำ (The Memory Hierarchy)
ในการออกแบบต้องมีการแลกเปลี่ยนกัน (**Trade-off**) ตามกฎ 4 ข้อ:
1. **ราคาต่อบิตลดลง** เมื่อลงไปในลำดับชั้นที่ต่ำกว่า
2. **ความจุเพิ่มขึ้น** เมื่อลงไปในลำดับชั้นที่ต่ำกว่า
3. **เวลาการเข้าถึงนานขึ้น (ช้าลง)** เมื่อลงไปในลำดับชั้นที่ต่ำกว่า
4. **ความถี่ในการเข้าถึง** โดยตัวประมวลผลลดลง
