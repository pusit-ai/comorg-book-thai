## สรุปคำสั่งที่ใช้บ่อย (Cheat Sheet)

1. `git pull` — ดึงงานล่าสุดจากเว็บ (ทำก่อนเริ่มงานเสมอ)
2. `git add .` — นำไฟล์ที่แก้ไขเข้าคิวเตรียมเซฟ
3. `git commit -m "ข้อความอธิบาย"` — บันทึกประวัติ
4. `git push` — ส่งงานขึ้นเว็บ
