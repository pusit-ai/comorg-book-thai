## จุดประสงค์

GitHub มีบริการแปลงไฟล์ Markdown (`.md`) ใน Repository ให้กลายเป็นหน้าเว็บไซต์ที่อ่านง่ายโดยอัตโนมัติ เพื่อให้นิสิตสามารถนำเสนอผลงานได้อย่างมืออาชีพ

---

## ขั้นตอนการเปิดใช้งาน

*(ดำเนินการโดยหัวหน้ากลุ่มบนเว็บ GitHub)*

1. เข้าไปที่หน้า Repository หลักของกลุ่ม (`comorg-book-thai`)
2. คลิกแท็บ **Settings** (สัญลักษณ์รูปฟันเฟืองด้านบนขวา)
3. ที่เมนูด้านซ้าย เลื่อนหาและคลิก **Pages**
4. ในหัวข้อ **Build and deployment** > **Source** เลือก **Deploy from a branch**
5. ในส่วน **Branch** เปลี่ยนจาก `None` เป็น `main` และเลือกโฟลเดอร์ `/ (root)`
6. กดปุ่ม **Save**
7. รอระบบประมวลผลประมาณ 1–2 นาที กดรีเฟรชหน้าเว็บ

---

## ผลลัพธ์

นิสิตสามารถนำลิงก์นี้ส่งอาจารย์ได้
